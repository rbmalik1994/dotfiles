macOS notes and platform-specific configuration

Keep macOS-specific settings, Homebrew package lists, and `defaults` commands here.

Suggested contents
- `macos/brew-packages.txt` — Homebrew formulae and casks
- `macos/defaults.sh` — macOS `defaults write` commands to apply settings

Install tip
-----------
Use Homebrew as the package manager and run `scripts/install.sh` on macOS to apply shared symlinks then run macOS-specific setup.
