Windows configuration and PowerShell setup notes

Use this folder to keep PowerShell profile, `winget` or `choco` package lists, and Windows-specific instructions.

Suggested contents
- `windows/packages.txt` — `winget` or `choco` install list
- `windows/Profile.ps1` — PowerShell profile to be linked to the user profile location

Install tip
-----------
Run `scripts/install.ps1` from an elevated PowerShell prompt after reviewing it.
